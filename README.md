# Sistema de Recomendação em Python

## Projeto desenvolvido no decorrer do curso "Inteligência Artificial - Sistema de recomendação" da Udemy

## Breve descrição
Trata-se de um curso em Python que aborda o desenvolvimento de algorítimos usados em um sistema de recomendação de filmes. Tais algorítimos têm como base o calculo da Distância Euclidiana para definir a semelhança dos perfis dos usuários de filmes. A partir dessa semelhança é feita a predição de notas que usuários dariam para filmes que ainda não assistiram e por fim, é feita a recomendação de filmes para esses usuários.

## Principais tecnologas utilizadas
